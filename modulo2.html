Módulo 2: Creación de proyectos y carga de datos

Aprende a crear tu primer proyecto en ATLAS.ti y a importar diferentes tipos de documentos para comenzar tu análisis cualitativo de manera organizada y eficiente.

Objetivos de aprendizaje:

  • Crear un nuevo proyecto en ATLAS.ti
  • Configurar las propiedades básicas del proyecto
  • Importar documentos de texto, imágenes, audio y video
  • Organizar y gestionar los documentos primarios
  • Establecer buenas prácticas para la estructura del proyecto

Creando tu primer proyecto

Un proyecto en ATLAS.ti es el contenedor principal de toda tu investigación. Es importante planificar bien la estructura desde el inicio para facilitar el trabajo posterior.

Pasos para crear un proyecto:

  1. Abrir ATLAS.ti: Inicia el programa desde tu escritorio o menú de aplicaciones
  2. Nuevo proyecto: Selecciona "File" > "New Project" o usa el atajo Ctrl+N
  3. Nombre del proyecto: Asigna un nombre descriptivo y significativo
  4. Ubicación: Elige una carpeta donde guardar el proyecto
  5. Configuración inicial: Define las propiedades básicas del proyecto
Recomendación: Usa nombres de proyecto que incluyan la fecha y una descripción breve, por ejemplo: "2025_Estudio_Comunicacion_Digital" o "Entrevistas_Profesores_Enero2025".

Configuración del proyecto

Antes de comenzar a cargar datos, es recomendable configurar algunos aspectos importantes:

Propiedades del proyecto

  • Título: Nombre completo de la investigación
  • Autor: Investigador principal o equipo
  • Descripción: Breve resumen del objetivo del estudio
  • Palabras clave: Términos relevantes para la investigación
  • Fecha de creación: Se asigna automáticamente

Configuración de respaldos

ATLAS.ti permite configurar respaldos automáticos para proteger tu trabajo:

Configuración recomendada:
• Activar respaldo automático cada 10-15 minutos
• Mantener al menos 5 versiones de respaldo
• Guardar respaldos en una ubicación diferente al proyecto principal

Importación de documentos

ATLAS.ti puede trabajar con múltiples tipos de documentos. La importación correcta es crucial para un análisis eficiente.

Documentos de texto

Los documentos de texto son los más comunes en investigación cualitativa:

  • Formatos soportados: PDF, DOC, DOCX, TXT, RTF
  • Preparación: Asegúrate de que el texto sea legible y esté bien formateado
  • Importación: Arrastra los archivos al área de documentos o usa "Documents" > "Add Documents"

Imágenes

Las imágenes pueden incluir fotografías, capturas de pantalla, diagramas, etc.:

  • Formatos soportados: JPG, PNG, GIF, BMP, TIFF
  • Resolución: Usa imágenes de calidad suficiente para el análisis
  • Organización: Agrupa imágenes relacionadas usando familias de documentos

Audio y video

Los archivos multimedia requieren consideraciones especiales:

  • Audio: MP3, WAV, WMA (asegúrate de tener buena calidad de sonido)
  • Video: MP4, AVI, MOV (considera el tamaño del archivo)
  • Transcripciones: Es recomendable tener transcripciones de texto asociadas
Importante: Los archivos multimedia pueden hacer que el proyecto sea muy pesado. Considera usar formatos comprimidos y mantén los archivos originales como respaldo.

Organización de documentos

Una buena organización desde el inicio facilitará el análisis posterior:

Familias de documentos

Las familias permiten agrupar documentos relacionados:

  • Por tipo: Entrevistas, observaciones, documentos oficiales
  • Por participante: Todos los documentos de una persona
  • Por tema: Documentos que abordan un tema específico
  • Por fecha: Documentos de un período temporal

Nomenclatura consistente

Establece un sistema de nombres claro y consistente:

Ejemplo de nomenclatura:
• ENT_001_Profesor_Universidad_A
• OBS_002_Clase_Comunicacion_Digital
• DOC_003_Plan_Estudios_2024
• IMG_004_Aula_Tecnologia

Gestión de documentos primarios

Una vez importados los documentos, puedes gestionarlos de varias maneras:

Propiedades de documentos

Cada documento puede tener metadatos asociados:

  • Nombre: Identificador único del documento
  • Comentario: Descripción o contexto del documento
  • Fecha: Fecha de creación o recolección
  • Autor: Quien creó o proporcionó el documento
  • Tipo: Categoría del documento

Filtros y búsquedas

ATLAS.ti ofrece herramientas para localizar documentos rápidamente:

  • Filtro por nombre: Busca documentos por su nombre
  • Filtro por familia: Muestra solo documentos de una familia específica
  • Filtro por tipo: Filtra por tipo de archivo (texto, imagen, audio, video)
  • Búsqueda de contenido: Busca texto específico dentro de los documentos

Buenas prácticas

Recomendaciones para la gestión de proyectos:
• Mantén una estructura de carpetas clara en tu computadora
• Haz respaldos regulares del proyecto completo
• Documenta tus decisiones metodológicas en memos
• Usa nombres descriptivos y consistentes
• Revisa y limpia regularmente los documentos no utilizados
• Considera el tamaño del proyecto para optimizar el rendimiento
